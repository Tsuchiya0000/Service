
# 創作支援ツール

創作活動をサポートするために作成したWebアプリです。  
パレット管理・お題生成・クリエイター管理などをまとめたツールキットです。

---

# できること

## パレットノート
- 色の保存
- パレット一覧表示
- 編集・削除
- JSONインポート・エクスポート
- ダークモード対応

---

## お題ジェネレーター
- ボタンを押すとランダムでお題を生成
- 創作アイデア出し用

---

## お気に入りクリエイター
- クリエイター情報の登録
- タグ管理
- 一覧表示

---

## HowTo（使い方）
- 各機能の使い方を説明するヘルプページ
- ポップアップ対応予定

---

# 技術構成

## フロントエンド
- HTML
- CSS
- JavaScript

---

## データ保存
- localStorageを使用
- パレットデータをJSON形式で保存

---

# データ構造

パレットデータは以下の形式で保存されます：

```
json id="palette_json_example"
{
    "name": "サンプルパレット",
    "colors": [
        "#ff0000",
        "#00ff00",
        "#0000ff",
        "#ffffff"
    ]
}
```
